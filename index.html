CS 61B Solo Project 2015-16

Text Editor

A project from Data Structures (CS 61B) designed to gain a understanding of data structures and how they are used.

Design

First I got familiar with the JavaFx library, what a text editor is, and the UI that was provided by the class. After understanding the basics I designed a data structure that I could use with UI.

Implementation

Originally I didn't account for all features that I wanted to implement and had to constantly alter my data structure in order to accomplish implementation of other features. I learned how to plan from this project, and how an unplanned project could halt progress.

What I Would do Differently Next Time

Before I start I would understand all features I would like my data structure to accomplish and include that in my system design phase. Having to go back and constantly add functionality in my data structure was very inefficient. Another problem I had was all of my code was completed on one page. Next time, I would make sure that I modularize all of my code to make debugging and understanding the code easier.
